# n8n-openai-image-generator
- n8n을 이용해 이미지를 생성해주는 workflow를 구축합니다.

## Tech Stack
- n8n
- OpenAI

## workflow
```
[n8n Form] - On new n8n Form event
↓
[OpenAI] - Generate an image
```
